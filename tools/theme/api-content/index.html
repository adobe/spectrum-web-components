Attributes and Properties

#Section titled Attributes and Properties
Property Attribute Type Default Description color color "lightest" | "light" | "dark" | "darkest" | "" The Spectrum color stops to apply to content scoped by this `sp-theme` element.

A value is required.

dir dir "ltr" | "rtl" | "" Reading direction of the content scoped to this `sp-theme` element. scale scale "medium" | "large" | "" The Spectrum platform scale to apply to content scoped by this `sp-theme` element.

A value is required.

system system "spectrum" | "express" The Spectrum system that is applied to the content scoped to this `sp-theme` element.

A value is required.
