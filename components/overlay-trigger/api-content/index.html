By explicitly declaring which content types are used (e.g. "click", "longpress hover"), we can avoid:

  1. Extra renders from unnecessary slot reparenting
  2. Potential infinite render loops during content detection
  3. Race conditions during slot assignment

By only returning overlay wrappers for explicitly declared content types, we minimize unecessary DOM nodes, operations and ensure a more stable rendering behavior.
