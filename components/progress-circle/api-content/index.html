Attributes and Properties

#Section titled Attributes and Properties
Property Attribute Type Default Description label label string '' Accessible label for the progress circle.

Used to provide context about what is loading or progressing. When no accessible name is provided (no label, aria-label, or aria-labelledby), a default "Loading" label is applied.

progress progress number | null null Progress value from 0 to 100.

When null (indeterminate), the component shows a loading animation. Setting a number switches to determinate mode. Removing the progress attribute or setting this property to null returns to indeterminate. Values outside 0–100 or non-finite numbers are clamped (non-finite becomes 0).

Reflected to the progress attribute when set; the attribute is omitted when indeterminate.

size ElementSize The size of the progress circle. staticColor static-color ProgressCircleStaticColorS1 | undefined Static color variant for use on different backgrounds.

When set to 'white', the component uses white styling for images with a dark tinted background.
