# sp-table-cell

An sp-table is used to create a container for displaying information.

```js
import '@spectrum-web-components/table/sp-table-cell.js';
// <sp-table-cell></sp-table-cell>
```

## Attributes

| Name   | Type     | Default      | Description |
| ------ | -------- | ------------ | ----------- |
| `role` | `string` | `'gridcell'` |             |
