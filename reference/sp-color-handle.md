# sp-color-handle

The sp-color-handle is used to select a color on an sp-color-area , sp-color-slider , or sp-color-wheel .

```js
import '@spectrum-web-components/color-handle/sp-color-handle.js';
// <sp-color-handle></sp-color-handle>
```

## Attributes

| Name       | Type      | Default                  | Description |
| ---------- | --------- | ------------------------ | ----------- |
| `disabled` | `boolean` | `false`                  |             |
| `focused`  | `boolean` | `false`                  |             |
| `open`     | `boolean` | `false`                  |             |
| `color`    | `string`  | `'rgba(255, 0, 0, 0.5)'` |             |
