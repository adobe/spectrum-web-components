sp-progress-bar

An sp-progress-bar is used to visually show the progression of a system operation such as downloading, uploading, processing, etc.

import '@spectrum-web-components/progress-bar/sp-progress-bar.js';
// <sp-progress-bar></sp-progress-bar>
