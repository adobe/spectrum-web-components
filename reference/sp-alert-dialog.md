# sp-alert-dialog

sp-alert-dialog displays important information that users need to acknowledge.

```js
import '@spectrum-web-components/alert-dialog/sp-alert-dialog.js';
// <sp-alert-dialog></sp-alert-dialog>
```

## Attributes

| Name      | Type                  | Default | Description |
| --------- | --------------------- | ------- | ----------- |
| `variant` | `AlertDialogVariants` | -       |             |
