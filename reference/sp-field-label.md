# sp-field-label

An sp-field-label provides accessible labelling for form elements.

```js
import '@spectrum-web-components/field-label/sp-field-label.js';
// <sp-field-label></sp-field-label>
```

## Attributes

| Name           | Type                          | Default | Description |
| -------------- | ----------------------------- | ------- | ----------- |
| `disabled`     | `boolean`                     | `false` |             |
| `id`           | `string`                      | `''`    |             |
| `for`          | `string`                      | `''`    |             |
| `required`     | `boolean`                     | `false` |             |
| `side-aligned` | `'start' | 'end' | undefined` | -       |             |

## Slots

| Name        | Description               |
| ----------- | ------------------------- |
| `(default)` | text content of the label |
