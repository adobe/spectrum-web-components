# sp-help-text

An sp-help-text provides either an informative description or an error message that gives more context about what a user needs to input.

```js
import '@spectrum-web-components/help-text/sp-help-text.js';
// <sp-help-text></sp-help-text>
```

## Attributes

| Name      | Type               | Default     | Description                                    |
| --------- | ------------------ | ----------- | ---------------------------------------------- |
| `icon`    | `boolean`          | `false`     |                                                |
| `variant` | `HelpTextVariants` | `'neutral'` | The visual variant to apply to this help text. |
