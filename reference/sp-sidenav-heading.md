# sp-sidenav-heading

Side navigation allows users to locate information and features within the UI.

```js
import '@spectrum-web-components/sidenav/sp-sidenav-heading.js';
// <sp-sidenav-heading></sp-sidenav-heading>
```

## Attributes

| Name    | Type     | Default | Description |
| ------- | -------- | ------- | ----------- |
| `label` | `string` | `''`    |             |

## Slots

| Name        | Description                                                  |
| ----------- | ------------------------------------------------------------ |
| `(default)` | the Sidenav Items to display in association with the heading |
