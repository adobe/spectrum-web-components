# sp-tabs-overflow

The sp-tabs displays a list of sp-tab element children as role="tablist".

```js
import '@spectrum-web-components/tabs/sp-tabs-overflow.js';
// <sp-tabs-overflow></sp-tabs-overflow>
```

## Attributes

| Name             | Type      | Default                     | Description |
| ---------------- | --------- | --------------------------- | ----------- |
| `compact`        | `boolean` | `false`                     |             |
| `label-previous` | `string`  | `'Scroll to previous tabs'` |             |
| `label-next`     | `string`  | `'Scroll to next tabs'`     |             |
