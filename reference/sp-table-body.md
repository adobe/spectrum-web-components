# sp-table-body

An sp-table is used to create a container for displaying information.

```js
import '@spectrum-web-components/table/sp-table-body.js';
// <sp-table-body></sp-table-body>
```

## Attributes

| Name   | Type     | Default      | Description |
| ------ | -------- | ------------ | ----------- |
| `role` | `string` | `'rowgroup'` |             |
