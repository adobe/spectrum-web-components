# `Button`

#### `loads with href`

```html
<a
  href="test_url"
  id="button"
  tabindex="0"
>
  <div id="label">
    <slot id="slot">
    </slot>
  </div>
</a>

```

#### `loads with href and target`

```html
<a
  href="test_url"
  id="button"
  tabindex="0"
  target="_blank"
>
  <div id="label">
    <slot id="slot">
    </slot>
  </div>
</a>

```
