# Color loupe migration guide

Replace `<sp-color-loupe>` with `<swc-color-loupe>`, update the import, and remove any `--mod-colorloupe-*` or `--spectrum-picked-color` CSS overrides.

## What changed

### Renamed

| Area        | Spectrum 1                                               | Spectrum 2                                  |
| ----------- | -------------------------------------------------------- | ------------------------------------------- |
| Tag name    | `sp-color-loupe`                                         | `swc-color-loupe`                           |
| Import path | `@spectrum-web-components/color-loupe/sp-color-loupe.js` | `@adobe/spectrum-wc/components/color-loupe` |

### Removed in Spectrum 2

| Removed                                       | Replacement                                                                               |
| --------------------------------------------- | ----------------------------------------------------------------------------------------- |
| `--spectrum-picked-color` CSS custom property | Use the `color` attribute (see [step 3](#3-set-the-picked-color-via-the-color-attribute)) |
| All `--mod-colorloupe-*` CSS override hooks   | None — the customization surface is intentionally removed                                 |

## Update your code

### 1. Update the import

```js
// Before
import '@spectrum-web-components/color-loupe/sp-color-loupe.js';

// After
import '@adobe/spectrum-wc/components/color-loupe';
```

### 2. Rename the tag

```html
<!-- Before -->
<sp-color-loupe open color="rgba(0, 128, 255, 0.7)"></sp-color-loupe>

<!-- After -->
<swc-color-loupe open color="rgba(0, 128, 255, 0.7)"></swc-color-loupe>
```

The `open` and `color` attributes work identically.

### 3. Set the picked color via the `color` attribute

If you were driving the displayed color through `--spectrum-picked-color` or a `--mod-colorloupe-*` property, switch to the `color` attribute (or `element.color` in JavaScript). All valid CSS color strings are accepted, including formats with alpha transparency.

```html
<!-- Before: setting color via CSS custom property -->
<style>
  sp-color-loupe {
    --spectrum-picked-color: rgba(0, 128, 255, 0.7);
  }
</style>
<sp-color-loupe open></sp-color-loupe>

<!-- After: set color directly on the element -->
<swc-color-loupe open color="rgba(0, 128, 255, 0.7)"></swc-color-loupe>
```

## Accessibility

The loupe is a visual-only, presentational element with no interactive role. No consumer-facing markup changes are required. Accessibility semantics must continue to be provided by the parent color picker control (`aria-label`, live region announcements, and so on) — the loupe itself does not receive focus and does not require ARIA attributes.
