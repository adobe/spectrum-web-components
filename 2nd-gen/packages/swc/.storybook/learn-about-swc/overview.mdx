# Overview

Spectrum Web Components (SWC) is Adobe's implementation of the Spectrum design system as [web components](https://developer.mozilla.org/docs/Web/Web_Components), built with [Lit](https://lit.dev/). This site documents the Gen2 release, which aligns components with [Spectrum 2](https://s2.spectrum.adobe.com/) on a new architecture focused on performance and accessibility.

Web Components are a set of technologies that work together, letting you create custom elements that work just like the standard HTML elements built into your browser.

You can use web components with any framework, since they’re supported and encapsulated at the browser level.

## What's new in Gen2

- Full-fidelity Spectrum 2 styling, tokens, and interaction behaviors
- Refreshed architecture with more consistent component APIs
- Improved accessibility coverage and bug fixes from Gen1
- Smaller bundles and faster runtime, with broader reliance on browser standards
- Active support: new components, features, and fixes

See [Gen1 vs Gen2](/docs/learn-about-swc-gen1-vs-gen2--docs) for a side-by-side comparison.
