# Core controllers

These pages document reactive controllers from `@spectrum-web-components/core/controllers`. Controllers implement cross-cutting behavior you can attach to Lit `ReactiveElement` hosts with `addController`.

## Available documentation

- **[Focus group navigation controller](../?path=/docs/controllers-focus-group-navigation-controller--readme)** — roving `tabindex`, arrow-key movement, and optional wrap/memory aligned with APG and the Open UI `focusgroup` explainer.

Import paths use the package exports, for example:

```typescript
import { FocusgroupNavigationController } from '@spectrum-web-components/core/controllers/focus-group-navigation-controller.js';
```
