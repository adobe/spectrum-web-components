# Accessible pattern libraries

This section collects patterns for building accessible UIs. Use it together with foundation guides: [Accessible landmarks](./accessible_landmarks.mdx) (page **landmarks** and `h1`–`h6` structure, **swc** heading APIs), [Semantic HTML and ARIA](./semantic_html_aria.mdx) (native elements vs ARIA, cross-root limits), and [WAVE toolbar testing](./wave_toolbar_testing.mdx) (quick **Structure** / **Order** / **Contrast** checks in the browser). For manual testing, see [Keyboard testing](./keyboard_testing.mdx) and [Screen reader testing](./screen_reader_testing.mdx). [Overview](./overview.mdx) lists all guides in this section.
