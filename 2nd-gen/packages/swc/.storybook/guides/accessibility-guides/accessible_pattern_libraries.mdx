# Accessible pattern libraries

This section collects patterns for building accessible UIs. Use it together with foundation guides: [Headings and landmarks](/docs/guides-accessibility-guides-headings-and-landmarks--readme) (page **landmarks** and `h1`–`h6` structure, **swc** heading APIs), [Semantic HTML and ARIA](/docs/guides-accessibility-guides-semantic-html-aria--readme) (native elements vs ARIA, cross-root limits), and [WAVE toolbar testing](/docs/guides-accessibility-guides-wave-toolbar-testing--readme) (quick **Structure** / **Order** / **Contrast** checks in the browser). For manual testing, see [Keyboard testing](/docs/guides-accessibility-guides-keyboard-testing--readme) and [Screen reader testing](/docs/guides-accessibility-guides-screen-reader-testing--readme). [Overview](/docs/guides-accessibility-guides-overview--readme) lists all guides in this section.
