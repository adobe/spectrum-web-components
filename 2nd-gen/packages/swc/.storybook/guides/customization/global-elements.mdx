# Global element styling

Optional styling is available for "global elements" such as button-styled native links. This is in an effort to retain native semantics and default accessibility benefits.

For global element styles to be fully available to slotted elements, you must include the stylesheet for `@adobe/spectrum-wc/global-elements.css` at the application level and ensure it's processed by your bundler to resolve the relative imports.

To use global element styles within the shadow DOM of extended and custom components, ensure their stylesheet also imports the global stylesheet.

Individual stylesheets are also available instead of the comprehensive index as `global-[component].css`.

## Customizing global elements

There are two ways to provide style customizations to global elements:

### Custom properties

The preferred method is to pass overrides to the exposed custom properties.

```css
.swc-Button {
  --swc-button-max-inline-size: 120px;
}
```

### Overrides cascade layer

If you need to modify properties not exposed through custom properties, you can pass your additions to the global element by including styles within the `swc-global-element-overrides` layer.

```css
@layer swc-global-element-overrides {
  .swc-Button {
    color: blue;
  }
}
```

## Button Styles

You may use button styling for either native links or button elements. Classes use BEM-style naming, and require the `swc-Button` class in addition to any variant classes.

**Note**: button-styled native links do not support disabled or pending states.

{/* prettier-ignore */}
<a href="#" class="swc-Button">As Link</a>
{/* prettier-ignore */}
<button type="button" class="swc-Button">As Button</button>

```html
<a href="#" class="swc-Button">As Link</a>
<button type="button" class="swc-Button">As Button</button>
```

Links and buttons support variant styles using associated modifier classes.

{/* prettier-ignore */}
<a href="#" class="swc-Button swc-Button--secondary">As Link (Secondary)</a>
<a href="#" class="swc-Button swc-Button--accent">As Link (Accent)</a>
<a href="#" class="swc-Button swc-Button--primary swc-Button--outline">As Link (Outlined)</a>
<button class="swc-Button swc-Button--staticBlack">As Button (Static Black)</button>

```html
<a href="#" class="swc-Button swc-Button--secondary">As Link (Secondary)</a>
<a href="#" class="swc-Button swc-Button--accent">As Link (Accent)</a>
<a href="#" class="swc-Button swc-Button--primary swc-Button--outline">
  As Link (Outlined)
</a>
<button class="swc-Button swc-Button--staticBlack">
  As Button (Static Black)
</button>
```

Buttons support `disabled` state styles.

{/* prettier-ignore */}
<button type="button" class="swc-Button" disabled>As Button (Disabled)</button>

```html
<button type="button" class="swc-Button" disabled>As Button (Disabled)</button>
```

Using a wrapper around the label with class `swc-Button-label` is recommended to best support text wrapping. It is required for truncation.

{/* prettier-ignore */}
<button type="button" class="swc-Button" style={{ '--swc-button-max-inline-size': '120px' }}><span class="swc-Button-label">Submit to stakeholders</span></button>
<button type="button" class="swc-Button swc-Button--truncate" style={{ '--swc-button-max-inline-size': '120px' }}><span class="swc-Button-label">Submit to stakeholders</span></button>

```html
<button
  type="button"
  class="swc-Button"
  style="--swc-button-max-inline-size: 120px"
>
  <span class="swc-Button-label">Submit to stakeholders</span>
</button>
<button
  type="button"
  class="swc-Button swc-Button--truncate"
  style="--swc-button-max-inline-size: 120px"
>
  <span class="swc-Button-label">Submit to stakeholders</span>
</button>
```

Icons in buttons or icon-only variants are supported, with the assumption of an SVG as the icon with the class of `swc-Button-icon`. For best text wrapping support, add `swc-Button--hasIcon` to the base element.

Icon-only should include the class `swc-Button--iconOnly` on the base element, and provide a label via `aria-label`.

{/* prettier-ignore */}
<button type="button" class="swc-Button swc-Button--hasIcon">
  <svg class="swc-Button-icon" xmlns="http://www.w3.org/2000/svg" viewBox="0 0 36 36" aria-hidden="true" focusable="false"><path d="M31.5 17H19V4.5a1 1 0 0 0-2 0V17H4.5a1 1 0 0 0 0 2H17v12.5a1 1 0 0 0 2 0V19h12.5a1 1 0 0 0 0-2z"/></svg>
  <span class="swc-Button-label">With Icon</span>
</button>
<button type="button" class="swc-Button swc-Button--iconOnly" aria-label="icon-only">
  <svg class="swc-Button-icon" xmlns="http://www.w3.org/2000/svg" viewBox="0 0 36 36" aria-hidden="true" focusable="false"><path d="M31.5 17H19V4.5a1 1 0 0 0-2 0V17H4.5a1 1 0 0 0 0 2H17v12.5a1 1 0 0 0 2 0V19h12.5a1 1 0 0 0 0-2z"/></svg>
</button>

```html
<button type="button" class="swc-Button swc-Button--hasIcon">
  <svg class="swc-Button-icon" aria-hidden="true" focusable="false">
    <!-- ... -->
  </svg>
  <span class="swc-Button-label">With Icon</span>
</button>

<button
  type="button"
  class="swc-Button swc-Button--iconOnly"
  aria-label="icon-only"
>
  <svg class="swc-Button-icon" aria-hidden="true" focusable="false">
    <!-- ... -->
  </svg>
</button>
```
