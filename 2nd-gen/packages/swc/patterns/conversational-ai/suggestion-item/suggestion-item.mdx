## Getting started

Add the package to your project:

```zsh
yarn add @adobe/spectrum-wc
```

Import the side-effectful registration of `<swc-suggestion-item>`:

```ts
import '@adobe/spectrum-wc/patterns/conversational-ai/suggestion-item/swc-suggestion-item.js';
```

To reference the `SuggestionItem` type, use a type-only import:

```ts
import type { SuggestionItem } from '@adobe/spectrum-wc/patterns/conversational-ai/suggestion-item';
```

> The class is exposed primarily for type purposes. Extending it is possible, but its internal shape is not part of the public API. Subclasses may require changes between releases.

## Accessibility

### Features

The `<swc-suggestion-item>` element implements:

- Native `<button>` interaction semantics
- Keyboard activation via <kbd>Enter</kbd> and <kbd>Space</kbd>
- Bubbled `swc-suggestion` custom event on activation
