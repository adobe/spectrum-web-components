## Getting started

Add the package to your project:

```zsh
yarn add @adobe/spectrum-wc
```

Import the side-effectful registration of `<swc-upload-artifact>`:

```ts
import '@adobe/spectrum-wc/patterns/conversational-ai/upload-artifact/swc-upload-artifact.js';
```

To reference the `UploadArtifact` type, use a type-only import:

```ts
import type { UploadArtifact } from '@adobe/spectrum-wc/patterns/conversational-ai/upload-artifact';
```

> The class is exposed primarily for type purposes. Extending it is possible, but its internal shape is not part of the public API. Subclasses may require changes between releases.

## Options

### Multi Artifact

Multiple **`swc-upload-artifact`** nodes in a wrapping flex row, similar width to the prompt-field composer attachment strip, so card vs media spacing and wraps are easy to compare in isolation.

To see the same tiles inside **`swc-prompt-field`**, open **[Artifact](/docs/paatterns-conversational-ai-prompt-field--readme#artifact)**.

<Canvas of={UploadArtifactStories.MultiArtifact} />

### Card

Card type uses a compact thumbnail with horizontal text layout.

<Canvas of={UploadArtifactStories.Card} />

### Media

Media type uses a larger preview region without title and subtitle text.

<Canvas of={UploadArtifactStories.Media} />

### Text Overflow

Title and subtitle truncate with an ellipsis when the artifact is narrower than the text. **Card** uses a compact row; **media** keeps the actions region visible while the title shrinks.
