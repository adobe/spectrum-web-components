## Anatomy

A response status indicator consists of:

1. **Status row**: an indeterminate progress circle (loading) or checkmark (complete) with a label
2. **Reasoning toggle**: optional expandable disclosure for chain-of-thought content

<Canvas of={ResponseStatusStories.Anatomy} />

## Options

### Loading

The `loading` attribute controls which indicator is shown:

- **`loading=true`**: indeterminate progress circle + default loading label
- **`loading=false`**: checkmark + default complete label
- Use `slot="label"` to customize row text

<Canvas of={ResponseStatusStories.Loading} />

### Reasoning

Set **`open`** to control reasoning disclosure:

- **`open=false`**: reasoning collapsed
- **`open=true`**: reasoning expanded

The disclosure chevron only appears when default-slot reasoning content exists. While **`loading=true`**, reasoning UI is not shown. If reasoning slot content is later removed, the component automatically resets `open` to `false`. This internal collapse does not emit `swc-response-status-toggle`.

<Canvas of={ResponseStatusStories.Reasoning} />

## Accessibility

### Features

The `<swc-response-status>` element implements the following accessibility features:

#### Status announcement

- The loading row carries `role="status"`, so the status label is announced for screen readers
- When reasoning content exists, the reasoning toggle uses `aria-expanded` and `aria-controls` to communicate panel state
- The reasoning panel uses `role="group"` with an accessible name from `slot="reasoning-label"` (defaults to ‚ÄúReasoning‚Äù)

<Canvas of={ResponseStatusStories.Accessibility} />

#### Agentic mode

In agentic mode (`phase` set and/or `swc-response-status-step` children), use slotted labels instead of multiple label attributes:

- **`slot="label"`** on `<swc-response-status>` ‚Äî header row label
- **`slot="summary"`** ‚Äî initiating copy before the first step roll (defaults to ‚ÄúProcessing request‚Äù)
- **`slot="reasoning-label"`** ‚Äî accessible name for the step list / reasoning panel (defaults to ‚ÄúReasoning‚Äù)
- **`slot="label"`** on each `<swc-response-status-step>` ‚Äî step title
- **Default slot** on each step ‚Äî step detail in the expanded timeline

A visually hidden live region announces **Response processing** at generation start and **Response complete** when finished ‚Äî not intermediate rolling titles.
