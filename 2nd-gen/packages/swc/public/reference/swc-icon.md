# swc-icon

Minimal icon renderer that accepts slotted SVG markup.

```js
import '@adobe/spectrum-wc/components/icon/index.js';
// <swc-icon></swc-icon>
```

## Attributes

| Name    | Type          | Default | Description                    |
| ------- | ------------- | ------- | ------------------------------ |
| `size`  | `ElementSize` | -       | T-shirt icon size.             |
| `label` | `string`      | `''`    | Accessible label for the icon. |
